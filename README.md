**Problem:** According to studies, over 60% of Indian youth cannot name all fundamental rights guaranteed by the Constitution.

**Our Solution:** 
- Interactive lessons breaking down complex constitutional concepts
- Real-world examples connecting rights to everyday situations
- Progressive difficulty levels ensuring comprehensive understanding

### 2. **Increasing Voter Awareness**
**Problem:** Low voter turnout and uninformed voting decisions affect democracy.

**Our Solution:**
- Educational modules on voting rights and electoral processes
- Civic responsibility awareness through engaging content
- Understanding the impact of civic participation

### 3. **Legal Literacy for Common Citizens**
**Problem:** Many citizens are unaware of basic laws affecting their daily lives (traffic rules, public conduct, property rights).

**Our Solution:**
- Practical lessons on everyday laws and regulations
- AI chatbot providing instant answers to civic queries
- Real-life scenarios teaching legal compliance

### 4. **Empowering Youth for Nation Building**
**Problem:** Disconnect between constitutional values and youth engagement.

**Our Solution:**
- Gamified approach making civic education appealing to younger generations
- Progress tracking and achievements system fostering continuous learning
- Community building through shared learning experiences

### 5. **Accessibility & Inclusivity**
**Problem:** Quality civic education is often limited to urban areas and expensive institutions.

**Our Solution:**
- Free, web-based platform accessible from anywhere
- Mobile-responsive design for smartphone users
- Simple, clear language avoiding legal jargon

## ğŸš€ Getting Started

### Prerequisites

- **Node.js** 18.x or higher
- **npm** or **yarn**
- **PostgreSQL** database (Supabase recommended)
- **Clerk** account for authentication
- **AI API Key** (Gemini)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/nitin864/TEAM-LOCALHOST_AMUHACKS5.0.git
cd TEAM-LOCALHOST_AMUHACKS5.0
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Set up environment variables**

Create a `.env.local` file in the root directory:

```env
# Clerk Authentication
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_xxxxx
CLERK_SECRET_KEY=sk_test_xxxxx

# Database (Supabase )
DATABASE_URL="your supase key"

# AI API 
 
NEXT_PUBLIC_GEMINI_API_KEY=xxxxx

# App URL
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

4. **Set up the database**
```bash
# Push schema to database
npm run db:push

# Seed initial data (optional)
npm run db:seed
```

5. **Run the development server**
```bash
npm run dev
```

6. **Open your browser**
```
http://localhost:3000
```

## ğŸ”§ Technical Implementation

### Key Technical Decisions

#### 1. **Next.js 16 with Turbopack**
- **Why**: 700x faster than Webpack, improved development experience
- **Benefits**: Hot reload in <1s, optimized production builds
- **Implementation**: App Router for modern routing, Server Components for performance

#### 2. **Server Actions over API Routes**
- **Why**: Type-safe, direct server mutations without API layer
- **Benefits**: Reduced latency, automatic request deduplication
- **Example**:
```typescript
"use server"
export async function upsertChallengeProgress(challengeId: number) {
  const { userId } = await auth();
  // Direct database mutation
}
```

#### 3. **Drizzle ORM**
- **Why**: Lightweight, type-safe, SQL-like syntax
- **Benefits**: Zero runtime overhead, excellent TypeScript support
- **vs Prisma**: 2x faster queries, smaller bundle size

#### 4. **Supabase Serverless PostgreSQL**
- **Why**: Auto-scaling, zero cold starts
- **Benefits**: Pay-per-use, instant branching for dev/staging
- **Cost**: ~$0 for small projects, scales infinitely

#### 5. **Clerk Authentication**
- **Why**: Drop-in auth with beautiful UI
- **Benefits**: OAuth providers, user management, security best practices
- **vs NextAuth**: Better UX, managed service, fewer bugs

#### 6. **Google Gemini for AI**
- **Why**: Powerful multimodal AI with generous free tier
- **Benefits**: Fast responses, strong reasoning capabilities, easy API integration, backed by Google
- **Model**: Gemini 1.5 Flash (optimized for fast civic Q&A and educational assistance)
